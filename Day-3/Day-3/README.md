# 📅 Day 3 – Arrays

## 📘 Topic
Arrays

## 💻 Problem Solved
**Check if an Array is Sorted**

## 📝 Problem Statement
Given an array, determine whether it is sorted in non-decreasing order.

Return:
- `true` if sorted
- `false` otherwise

---

## 🧠 Approach

- Traverse the array from index `1`.
- Compare each element with its previous element.
- If the current element is smaller than the previous element, return `false`.
- If the loop completes successfully, return `true`.

---

## 💡 Key Learning

- Learned how to compare adjacent elements.
- Understood early termination using `return false`.
- Strengthened array traversal concepts.
- Learned to identify sorted arrays efficiently.
