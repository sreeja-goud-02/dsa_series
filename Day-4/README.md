# Day 4 - Remove Duplicates from Sorted Array

## Problem
LeetCode #26 - Remove Duplicates from Sorted Array

## Language
Java

## Approach
- Used Two Pointer Technique.
- Compared current element with the last unique element.
- Updated the array in-place.

## Time Complexity
O(n)

## Space Complexity
O(1)
